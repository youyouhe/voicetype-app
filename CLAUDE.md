# CLAUDE.md\n\nThis file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.\n\n## Commands\n\n### Frontend (Vite/TS)\n- `npm run dev`: Start Vite dev server (localhost:1420)\n- `npm run build`: TypeScript check + Vite build (outputs to dist/)\n- `npm run preview`: Preview production build\n\n### Tauri\n- `npm run tauri dev`: Run Tauri dev (uses npm run dev + devUrl: http://localhost:1420)\n- `npm run tauri build`: Tauri build (uses npm run build + frontendDist: ../dist)\n\n### Rust\n- `cargo check`: Fast Rust compilation check\n- `cargo test`: Run Rust unit tests\n\n## Architecture\n\nStandard Tauri v2 desktop app:\n\n- **Frontend**: Vanilla TypeScript + Vite. Entry: `index.html` → `src/main.ts` (invokes Rust via `@tauri-apps/api/core`).\n- **Backend**: Rust (`src-tauri/src/lib.rs`) with commands `greet(name: &str) → String` and `add(a: i32, b: i32) → i32`. Uses `tauri_plugin_opener`.\n  Entry: `lib.rs::run()` → `tauri::Builder` → `invoke_handler![greet, add]`.\n- **Integration**: Tauri loads Vite dev server in dev mode; bundles `dist/` in build.\n- **ui/**: Separate React/Vite project (unrelated to main Tauri app).\n\nKey files:\n- Config: `package.json`, `src-tauri/tauri.conf.json`, `vite.config.ts`, `src-tauri/Cargo.toml`\n- Frontend: `src/main.ts`, `index.html`\n- Backend: `src-tauri/src/lib.rs`\n
